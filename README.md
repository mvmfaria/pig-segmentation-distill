## Setting up the dataset (PigLife)

1. Request access to the dataset at [PigLife - AIFARMS Data Portal](https://data.aifarms.org/view/piglife).
2. After you receive the download link, save the file as `.env.example` and then rename it to `.env`.
3. Make sure you have enough free disk space. The dataset is about **16 GB**.
4. Run `uv run inv build`
